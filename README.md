# 学校生活チャレンジ

`仕様.txt` をもとにした、学校生活向けソーシャルスキルトレーニングの Web アプリです。

## 構成

- `index.html`: 画面テンプレートと SPA の土台
- `styles.css`: 大きいボタンと視認性を重視したUI
- `app.js`: 面データ、進行制御、設定保存、カメラ / 音声連携
- `start-server.bat`: ローカル確認用の簡易サーバー起動
- `仕様.txt`: 元仕様

## 実装した内容

- ホーム、れんしゅう一覧、プレイ、カメラ実行、チャレンジ、設定の各画面
- 第1面〜第4面のシナリオデータ
- れんしゅう一覧から入れる第5面相当の `れんぞくチャレンジ`
- 2択選択と `やってみる` による進行
- マイク音量による簡易音声判定
- カメラを使った簡易 `とまる / みる` 判定
- 教師補助ボタンによる確実な判定
- `localStorage` を使った設定とベストコンボの保持

## 起動方法

カメラとマイクを使う場合は、`file:///.../index.html` ではなく `http://localhost` で開いてください。

### かんたんな方法

1. `start-server.bat` をダブルクリックします。
2. コマンド画面に `Serving HTTP on` と表示されたままにします。
3. ブラウザで `http://localhost:8000` を開きます。
4. カメラ / マイクの利用確認が出たら `許可` にします。

### 手動で起動する方法

```powershell
cd "C:\Users\templ\Documents\Codex\学校生活チャレンジ 行動でまなぶソーシャルスキル"
python -m http.server 8000
```

## 使い方

1. `http://localhost:8000` でアプリを開きます。
2. `れんしゅう` で第1面〜第5面を選ぶか、`チャレンジ` から開始します。
3. `せってい` で判定方式、ヒント、カウントダウン、必要コンボ数を調整できます。

## 補足

- `AI サポート` は現時点では簡易判定です。`はなす` は音量、`とまる` は動きの少なさ、`みる` は対応ブラウザでの顔位置を見ています。
- `みる` の自動判定はブラウザの `FaceDetector` に依存します。未対応環境では教師補助ボタンにフォールバックします。
- サーバー停止は、起動したコマンド画面で `Ctrl + C` を押します。
